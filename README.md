# Generere pensum

1. Lag et notat for kapittelet du jobber med i docs/notes/Kapittel_1.md
2. Lag finpusset notat i docs/begrep/Kapittel_1.md
3. Kjør `npm i && npm run generate` når du vil generere notater til eksamen
